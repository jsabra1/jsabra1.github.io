Dominic

What prevents the wrong classification of ads appearing on lower hierarchy channels in MRM?

Is there a warning that flashes up if you try to place an alcohol ad in children’s content, or does the system assume you know what you are doing?

  • No warning message is provided, however with forecasting you can see if a campaign is at risk of not serving because of any existing restrictions that might cause specific ads not to be placed on specific content.

 

Not only does MRM forecasting look back in order to predict future behaviour, but it takes into account MRM decisioning logic and all supported business dependencies, such as: targeting, restrictions etc.. Read the full article on Forecasting here: Forecasting Overview - MRM User Guide - Global Site (freewheel.tv) and more details on Transactional Forecasting here: MRM Transactional Forecasting - MRM User Guide - Global Site (freewheel.tv)

 

Additionally using restrictions hierarchy you can prevent the wrong classification of content to be displayed on a lower channel in the hierarchy, there are also options to override this.

You could see in the screenshot below the relationships that exist within the restrictions section (ancestors & descendants) as well as the type of inheritance that is set on the relevant item.

 

 

Tariq

How can you edit cue points on a video asset?

  • Cue points are ingested via Instant Video Ingest (IVI) which allows customers to import their video asset details into MRM via API, these are usually pre-defined by the content owner during such ingest or within the ad request when they have not been pre-defined. You can read more about IVI here: Instant Video Ingest (IVI) User Guide - TechDocs - Global Site (freewheel.tv)

 

How do you manage a clash of CBPs in MRM? - how do they get prioritised?

  • In the scenario where there are conflicting CBPs defined on the same level of a network item, the user will get a warning if there’s a clash and they can set precedence order for the conflicting CBPs.

 

Users can also define at the network setting level what priority to follow for the different network dimension.
 

Generally override and inherence settings can dictate which CBP is selected, if they are set to inherit then it would be the most restrictive settings that are applied to the lowest level.

 

You can read more on how conflicts between CBPs are resolved here: Investigating Commercial Break Pattern (CBP) Conflicts - MRM User Guide - Global Site (freewheel.tv)

 

 

How can we connect MRM to our external systems like our OMS?

  • This could be achieved via API integration, MRM offers extensive API documentation to integrate with your existing systems. You can find the API docs here: FreeWheel's Publisher API Documentation

 

Kevin

Can MRM Video Assets contain multiple external IDs so that different endpoints can each reference those different IDs for the same video asset?

  • Yes ingested video assets can have multiple Secondary Network External Asset ID's which will allow them to be identified by various endpoints. More information on how you can ingest your video assets into MRM via API calls here: Instant Video Ingest (IVI) User Guide - TechDocs - Global Site (freewheel.tv)

 

 

How can I view all the UEX controls on all levels at the same time rather than going into individual the levels?

  • Yes, if you go to the insights section under the Network tab, you'll be able to view all CPS, Restrictions and UEX caps that exist in your network.

 

 

 

 

Additionally, you can do this for individual groups of Network dimensions, for example if you look at a video or site section and switch to the Relationships tab view you can see where UEX controls are applied all the way up the tree.

 

 

Jack

How can attributes of users be uploaded and targeted in MRM?

  • This could be done using the user data ingest, ingested user will have a unique identifier and their data includes mapping to audience segments which allows the ability to target various users based on their audience preferences, you can read more about this here: How to Ingest Audience User Records - MRM User Guide - Global Site (freewheel.tv).

 

How do I test changes on a campaign before going live/ if the campaign is not working how to troubleshoot? 

  • Customers can use transactional forecasting to simulate changes to a campaign, some updates are on-demand and some are nightly. There are multiple factors that go into consideration when using forecasting and will give you an idea of how a campaign is going to perform given these factors:
    • Targeting criteria
    • CBP, UEX
    • Reseller ban
    • Advertiser blocklist/allowlist
    • Slot duration, ad duration

You can read more on MRM's Transactional Forecasting here: MRM Transactional Forecasting - MRM User Guide - Global Site (freewheel.tv)

 

Fred

What's the diff between UEX caps and Frequency caps

  • UEX caps have various controls and can be applied on network, video group, video and site level or even audience, 

Frequency caps can control how many times the same video will show with respect to the content and can be applied on the Campaign, Insertion Order, or Placement. New Campaign Tab - Setting Frequency Caps - MRM User Guide - Global Site (freewheel.tv)

During my experience working in recruitment & recruitment marketing as part of my part time business in 2018/19, I’ve had the pleasure to work with several employers & have spoken to many job seeking engineers, during that time I picked a few things. Thus, my ideas below are more targeted towards hiring Engineers or Tech talent in the UK which as you might already know are the toughest to find considering the great resignation we are witnessing across various markets. 

 

International Candidates, an Untapped Talent Pool (UK Specific)

The major change in perspective which many Big Tech companies have been implementing & benefiting from is hiring international talent, the war for talent has reached its peak in the wake of recovering from COVID, with many UK employers ramping up their hiring efforts & fiercely competing to hire/hunt from the same talent pool which is already saturated combined with the fact no new candidates are entering the market, the talent crisis continues. 

 

Luckily, the UK government has introduced less stringent immigration measures that allow employers to hire international professionals at various levels and across a wide range of skillsets & roles that even extend beyond Tech, such as Product Management, Sales, Support etc.. The introduction of the PBS (points-based system) allows us to hire from this pool without having to do the Labour Market Test (advertise for the roles for a certain duration of time) or jump through many other admin hoops. This isn’t considered common knowledge yet as only a few employers have adapted to this change and been able to benefit from it while the majority still struggle to fulfil their hiring needs. 

 

Employee experience & Employer Brand

Those have become a bit of buzzwords, however there are practical steps that could be taken to genuinely improve the way FreeWheel is seen as an Employer. 

 

For instance, in the of Software Development market, there’s been a major trend where many candidates are going into contracting as that offered them better pay & more freedom, which has left a limited number of candidates willing to take on permanent roles this meant employers were expected to introduce more attractive benefits & perks for perm roles such to attract such candidates and with Brexit & Covid this “nomad” trend has extended to many other roles surrounding Tech which can be done remotely.. One major topic we have been discussing at FW & Comcast is remote working & thanks Antony for pushing this conversation forward. If remote working aboard or at least across the country where each office is based (outside of covid) were to be considered, this would greatly enhance our ability to attract a whole new class of candidates.  

 

In addition to this, incentives to enhance the employee experience (please see list below) can go a long way in retaining & attracting returning talent. My ex-employer Avanade (a Microsoft Company) combined this with an “Associate Community” where people who left the company could still be part of that community & network with employees which created a lot of opportunities for many of them to return and work for the company. 

 

Some examples of benefits to consider: 

 

👩‍💻 Remote-first processes (or an office if you're in London).

🔋 Personal growth fund.

💻 New equipment and budget to build your own workspace (MacBook Pro or equivalent, monitors, software etc.)

⛰️ Quarterly Team offsite activities.

🏝️ Unlimited holiday (min 5.6 weeks).

🏃‍♀️ Monthly health stipend (£60 a month contributed to health related spend).

 

Enhancing Our TA Capabilities & Being Informed 

Implementing these or any improvements to our TA processes will require a strong team to drive this through. Sophie has been doing amazing work in hiring and I saw there was a new contractor onboarded recently to help her, however, at this scale it seems like a challenge for the existing team to keep up with such demand especially considering this will demand will continue to increase, thus the team can greatly benefit from enablement & strengthening by hiring permanent Technical Recruiters or Sourcers who are able to pre-screen candidates for roles in order to streamline the hiring process. 

 

More importantly expanding the team’s knowledge & awareness in the Skilled Worker Visa process mentioned above as that is a crucial part if that approach is to succeed & return results.
